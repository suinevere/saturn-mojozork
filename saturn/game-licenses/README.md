# Game licenses

The Z-Machine v3 story files under `../cd/data/Z3/` are Infocom's compiled games,
collected from Andrew Plotkin's [Obsessively Complete Infocom
Catalog](https://eblong.com/infocom/).

Starting **November 2025**, Microsoft began declaring Infocom games open source and
adding an MIT License to their source packages — first Zork 1, Zork 2, and Zork 3,
and since then **many more**. The source (and its MIT License) for each open-sourced
title lives in the [`historicalsource`](https://github.com/historicalsource)
collection on GitHub, e.g.:

- [historicalsource/zork1](https://github.com/historicalsource/zork1)
- [historicalsource/zork2](https://github.com/historicalsource/zork2)
- [historicalsource/zork3](https://github.com/historicalsource/zork3)
- [historicalsource/sorcerer](https://github.com/historicalsource/sorcerer)
- …and more added over time — check the org for the current list.

[`LICENSE-MIT.txt`](LICENSE-MIT.txt) here is that MIT License text (Copyright ©
2025 Microsoft), verbatim from those packages. It applies to every bundled game
Microsoft has open-sourced; for a specific game, confirm at its
`historicalsource/<game>` repo. Titles not yet declared open source are included
as-is from the catalog above for preservation/personal use.
